This project implements a cache/memory hierarchy with L1, L2, and L3 caches backed by a DRAM module. All caches are set associative: L1 is 2-way, L2 is 4-way, and L3 is 8-way. L1 uses true LRU, L2 uses pseudo-LRU, and L3 uses FIFO replacement policies. Caches use a write-back and write-allocate policy. Each line is 64 bytes and words are 4 bytes, meaning each line can store up to 16 words. Structured interfaces (cache_req_t and cache_resp_t) are used for communication between levels. Replacement policies are implemented in separate modules and integrated into the caches. DRAM is modeled behaviorally for simulation and testing.
